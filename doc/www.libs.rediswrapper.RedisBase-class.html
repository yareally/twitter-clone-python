_set_key_string(primary_key, secondary_key)
Static Method

source code 

Binds lookup keys to a string to query in redis. Examples: __set_key_string('user', 'id', email') returns 'user:id:email' __set_key_string('user', 'id') returns 'user:id'

Parameters:
  • primary_key - required main key to look up or set a value in redis (ex: 'user')
  • secondary_key - required subkey to look up or set a value in redis (ex: 'id')
Returns: str
The previous entry for the key being set or None if no previous entry exists

  • acquire_timeout (int) - time in seconds to wait for the lock to be set or time out if not
